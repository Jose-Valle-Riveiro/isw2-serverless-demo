# Reflexión
___

## 1. ¿Qué tipo de error evita el CI?
- Errores de construcción
- Errores de pruebas
- Calidad de código
- Errores de seguridad

## 2. ¿Qué tipo de error no evita?
- Errores de lógica de negocio
- Problemas de UX
- Vulnerabilidades

## 3. ¿Que pasaría si un equipo ignora el CI?
- Se pierde una de las herramientas más efectivas para detectar futuros errores
- Los errores se detectarían en etapas más tardias del desarrollo, donde son más costosos.
